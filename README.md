# Finance Tracker

A full-stack application to track your income and expenses.

## Project Overview

This application helps users manage their personal finances. It provides features for tracking transactions, viewing summaries, and analyzing spending habits. The frontend is built with React and the backend is a Python FastAPI server.

## Tech Stack

-   **Frontend:** React (Vite), Tailwind CSS, Recharts, TanStack Query, React Hook Form
-   **Backend:** Python, FastAPI, SQLAlchemy (async), PostgreSQL, Alembic
-   **Authentication:** JWT (JSON Web Tokens)
-   **Icons:** lucide-react

## Features

-   User registration and login
-   JWT-based authentication
-   Dashboard with a summary of income, expenses, and net balance
-   Filter dashboard summary by date range
-   Add, edit, and delete transactions
-   View a paginated list of all transactions
-   Search transactions by title
-   Upload a profile photo
-   View reports and analytics on spending
-   Responsive design for desktop and mobile

## Prerequisites

-   Node.js (v18 or higher)
-   Python (v3.10 or higher)
-   PostgreSQL

## Backend Setup

1.  **Navigate to the backend directory:**
    ```bash
    cd backend
    ```

2.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    ```

3.  **Activate the virtual environment:**
    -   On Windows:
        ```bash
        .\venv\Scripts\activate
        ```
    -   On macOS/Linux:
        ```bash
        source venv/bin/activate
        ```

4.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

5.  **Create a `.env` file:**
    Copy the contents of `.env.example` (if it exists) or create a new `.env` file with the following variables:
    ```
    DATABASE_URL="postgresql+asyncpg://<YOUR_DB_USER>:<YOUR_DB_PASSWORD>@localhost/<YOUR_DB_NAME>"
    SECRET_KEY="<YOUR_SECRET_KEY>"
    ALGORITHM="HS256"
    ACCESS_TOKEN_EXPIRE_MINUTES=30
    ```

6.  **Run database migrations:**
    ```bash
    alembic upgrade head
    ```

## Frontend Setup

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

## Running the App

1.  **Start the backend server:**
    From the `backend` directory, run:
    ```bash
    uvicorn app.main:app --reload
    ```
    The backend will be running at `http://localhost:8000`.

2.  **Start the frontend development server:**
    From the `frontend` directory, run:
    ```bash
    npm run dev
    ```
    The frontend will be running at `http://localhost:5173`.
